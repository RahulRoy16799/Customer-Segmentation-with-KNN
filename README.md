# Customer-Segmentation-with-KNN
This project performs customer segmentation using the Mall Customers dataset by applying K-Means clustering on different feature combinations. The analysis focuses on identifying patterns in Annual Income, Spending Score, and Age to group customers into meaningful segments.

K-Means Clustering:

🔹 1. Income vs Spending Score
Applied K-Means with 5 clusters
Visualized clusters with centroids
Identified distinct customer groups based on spending behavior

🔹 2. Age vs Spending Score
Applied K-Means with 4 clusters
Helped analyze how spending varies across different age groups
Revealed behavioral differences between younger and older customers

🔹 3. K-Means using 3 Features
Features used:
Age
Annual Income
Spending Score
Applied K-Means with 6 clusters
Created a new cluster column for segmentation

Dataset Link: https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python/data
